ProyectoU — CMS para Hogares Geriátricos

Finalizado React 19 Node.js Express 5 MongoDB Vite

Descripción

Sistema de Gestión de Contenido (CMS) multi-tenant especializado para la creación y administración de sitios web de hogares geriátricos. Permite a los usuarios crear, personalizar y exportar sitios web completos sin necesidad de conocimientos técnicos.

Desarrollado como proyecto universitario en la Universidad Libre Pereira, este sistema ofrece una plataforma modular donde cada sección de la página web es un módulo independiente con su propio editor visual, API endpoint y controlador. Todo el contenido se almacena en MongoDB y se renderiza dinámicamente en el frontend.

Características principales

  • ✓ Sistema multi-tenant: Múltiples usuarios, cada uno con su propio sitio web independiente
  • ✓ 10 módulos editables: Hero, servicios, galería, valores, equipo, video, mapa, carrusel y más
  • ✓ Edición visual sin código: Panel de administración intuitivo con vista previa en tiempo real
  • ✓ Personalización completa: Colores, fuentes, bordes y diseño totalmente personalizables
  • ✓ Exportación a sitio estático: Genera archivo ZIP con HTML + CSS + imágenes listo para deploy
  • ✓ Autenticación segura: JWT con sesiones de 7 días y hash bcrypt para contraseñas
  • ✓ Gestión de imágenes: Subida y almacenamiento de imágenes con validación de tipo y tamaño
  • ✓ Diseño responsive: Bootstrap 5 garantiza compatibilidad en todos los dispositivos

Stack tecnológico

Backend

  • • Node.js
  • • Express 5.2.1
  • • MongoDB + Mongoose 9.0.0
  • • JWT (jsonwebtoken 9.0.2)
  • • Bcrypt 6.0.0
  • • Multer 2.0.2
  • • Archiver 7.0.1
  • • CORS 2.8.5

Frontend

  • • React 19.2.3
  • • Vite 7.3.1
  • • React Router DOM 7.12.0
  • • Axios 1.13.2
  • • Bootstrap 5.3.3
  • • Bootstrap Icons
  • • Google Fonts (Poppins)

Arquitectura

Aplicación Full-Stack MERN con arquitectura modular:

  • → Frontend (React): SPA con Vite, Context API para autenticación, y Bootstrap 5 para UI responsive
  • → Backend (Node.js): API REST con Express, JWT para auth, Multer para archivos, Archiver para ZIP
  • → Base de datos: MongoDB con modelos User y PageConfig (almacena todas las secciones por usuario)

Funcionalidades destacadas

Panel de Administración

Interfaz intuitiva con tarjetas para cada módulo, vista previa en tiempo real y navegación fluida

Editor Visual

Sistema de edición en dos columnas: formulario a la izquierda y vista previa en vivo a la derecha

Exportación Inteligente

Genera sitio estático completo con HTML, CSS optimizado e imágenes descargadas en paralelo

Pantalla de Bienvenida

Landing page inmersiva con animaciones, partículas flotantes y gradientes cálidos

Sistema de Temas

ThemeProvider que aplica CSS variables dinámicas según la configuración del usuario

Integración WhatsApp

CTA de contacto con enlace directo a WhatsApp y mensajes pre-rellenados personalizables

Comfi Bot — Chatbot IA para Soporte Interno

En desarrollo Google Apps Script OpenAI API Firebase Firestore Google Sheets Google Chat API

Descripción

Comfi es un chatbot conversacional desarrollado en Google Apps Script e integrado directamente en Google Chat, diseñado para el departamento de Sistemas de Comfamiliar Risaralda. El bot recibe solicitudes de soporte de los colaboradores, las interpreta mediante inteligencia artificial (GPT-4o-mini de OpenAI) y las enruta automáticamente al agente disponible del área correspondiente.

El sistema utiliza un enrutamiento de dos capas: la IA analiza el mensaje y determina el área y tipo de soporte requerido, y un algoritmo de round-robin asigna el caso al agente óptimo según disponibilidad. Los casos quedan registrados en Firebase Firestore y sincronizados en Google Sheets, donde el equipo puede hacerles seguimiento.

La configuración de áreas y personal se gestiona desde una hoja de cálculo, lo que permite actualizar el sistema sin tocar el código.

Características principales

  • ✓ Enrutamiento inteligente con IA: GPT-4o-mini analiza el mensaje y determina automáticamente el área y tipo de soporte
  • ✓ Asignación por round-robin: Algoritmo que distribuye casos equitativamente entre agentes disponibles
  • ✓ Registro en tiempo real: Todos los casos se almacenan en Firebase Firestore con timestamps y metadata completa
  • ✓ Sincronización con Google Sheets: Dashboard automático para seguimiento y análisis de casos del equipo
  • ✓ Configuración dinámica: Áreas y personal se gestionan desde hojas de cálculo sin modificar código
  • ✓ Integración nativa: Funciona directamente en Google Chat sin instalaciones adicionales
  • ✓ Respuestas contextuales: El bot entiende lenguaje natural y proporciona confirmaciones personalizadas

Flujo de trabajo

  1. Colaborador envía mensaje de soporte en Google Chat
  2. Comfi Bot recibe el mensaje vía webhook
  3. El mensaje se envía a GPT-4o-mini para análisis
  4. La IA determina el área (redes, software, hardware, etc.) y el tipo de caso
  5. Algoritmo round-robin selecciona al agente disponible óptimo
  6. Se crea el registro en Firestore con toda la metadata
  7. Se sincroniza automáticamente en Google Sheets
  8. El bot notifica al colaborador con confirmación personalizada

Ventajas del sistema

Automatización total

Elimina la asignación manual de casos, reduciendo tiempos de respuesta

Distribución equitativa

Round-robin garantiza carga balanceada entre todos los agentes

Trazabilidad completa

Registro permanente en Firestore para auditorías y análisis

Configuración flexible

Personal y áreas se actualizan desde hojas sin tocar código

Estado del proyecto

Comfi Bot está actualmente en desarrollo activo como parte de mi pasantía en el departamento de Sistemas de Comfamiliar Risaralda. El sistema ya está operativo y procesando solicitudes reales del equipo interno.

Por tratarse de un proyecto interno corporativo, el código fuente no está disponible públicamente. Sin embargo, este caso de uso demuestra la aplicación práctica de inteligencia artificial para automatizar procesos de soporte técnico en entornos empresariales.
